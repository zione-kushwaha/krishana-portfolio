## ğŸ¯ Image Specifications

### 1. Profile Picture (`profile.png`)
- **Size:** 800x800px (minimum)
- **Format:** PNG with transparent background OR circular crop
- **Quality:** High resolution, well-lit professional photo
- **Tips:** 
  - Use a solid color background or blur it
  - Professional attire recommended
  - Good lighting is crucial
  - Smile naturally!

**Where to get:**
- Take a professional photo
- Use [Remove.bg](https://www.remove.bg/) to remove background
- Or use [PhotoRoom](https://www.photoroom.com/) for AI background removal

### 2. Hero Banner (`hero_banner.png`)
- **Size:** 1920x1080px
- **Format:** PNG or JPG
- **Ideas:**
  - Engineering blueprints overlay
  - Construction site panorama
  - Modern architecture
  - Abstract geometric patterns

**Free Resources:**
- [Unsplash](https://unsplash.com/) - Search: "engineering", "architecture", "blueprint"
- [Pexels](https://www.pexels.com/) - Search: "construction", "civil engineering"

### 3. Background Images
Create visual interest with subtle backgrounds:

#### Geometric Patterns (`backgrounds/geometric_*.png`)
- **Size:** 1920x1080px
- **Style:** Abstract, minimalist, modern
- **Tools:** 
  - [Haikei](https://haikei.app/) - Generate SVG backgrounds
  - [BGJar](https://bgjar.com/) - Free SVG backgrounds

#### Wave & Blob SVGs
- **Format:** SVG for scalability
- **Use:** Section dividers, decorative elements
- **Tools:**
  - [GetWaves](https://getwaves.io/)
  - [Blobmaker](https://www.blobmaker.app/)

### 4. Skill Icons

#### Technology Icons
Download high-quality SVG icons:
- **Flutter:** [flutter.dev](https://flutter.dev/brand)
- **AutoCAD, Revit, STAAD Pro:** Search on [Flaticon](https://www.flaticon.com/)
- **Generic Tech:** [Simple Icons](https://simpleicons.org/)

#### Social Media Icons
- **Recommended:** [Iconscout](https://iconscout.com/icon-pack/social-media-92)
- **Format:** SVG
- **Style:** Consistent (all outlined OR all filled)
- **Color:** Monochrome (app will colorize them)

### 5. Project Images

#### Thumbnails (`projects/project_*/thumbnail.jpg`)
- **Size:** 1200x800px
- **Format:** JPG (optimized)
- **Content:**
  - Main project photo
  - Before/After comparisons
  - 3D renders if available
  - Construction progress

#### Detail Images
- **Size:** 1920x1080px
- **Format:** JPG
- **Include:**
  - Multiple angles
  - Close-up details
  - Technical drawings
  - Final results

**Tips for Project Photos:**
- Use good lighting
- Clean, professional compositions
- Include scale references (people, objects)
- Highlight unique features

### 6. Illustrations

Free high-quality illustration resources:
- [unDraw](https://undraw.co/) - Customizable color illustrations
- [Storyset](https://storyset.com/) - Animated illustrations
- [Blush](https://blush.design/) - Mix & match illustrations
- [DrawKit](https://www.drawkit.com/) - Free SVG illustrations

**Recommended Illustrations:**
- Engineering/Technical work
- Teamwork/Collaboration
- Innovation/Ideas
- Success/Achievement
- Problem-solving

### 7. Patterns & Textures

Subtle patterns for card backgrounds:
- **Opacity:** 5-10% (very subtle!)
- **Colors:** Monochrome or matching theme
- **Tools:**
  - [Subtle Patterns](https://www.toptal.com/designers/subtlepatterns/)
  - [Hero Patterns](https://heropatterns.com/)
  - [Pattern.css](https://bansal.io/pattern-css)

## ğŸ“ Optimization Guidelines

### Before Adding Images:

1. **Compress Images:**
   - Use [TinyPNG](https://tinypng.com/) or [Squoosh](https://squoosh.app/)
   - Target: < 500KB for photos, < 100KB for icons
   - Maintain quality above 80%

2. **Use Correct Formats:**
   - **Photos:** JPG (smaller size)
   - **Graphics with transparency:** PNG
   - **Icons & logos:** SVG (scalable, tiny size)
   - **Animations:** WebP or SVG

3. **Responsive Images:**
   - Provide 1x, 2x, 3x versions for different screen densities
   - Or use SVG wherever possible

## ğŸ’¡ Pro Tips

1. **Consistency is Key:**
   - Use similar lighting/style for all photos
   - Stick to your color palette
   - Match illustration styles

2. **Less is More:**
   - Don't overcrowd with too many images
   - Use white space effectively
   - Focus on quality over quantity

3. **Performance Matters:**
   - Compress all images before adding
   - Use lazy loading for below-fold images
   - Consider WebP format for better compression

4. **Brand Identity:**
   - Create a simple logo/monogram
   - Use consistent filters on photos
   - Maintain professional tone
